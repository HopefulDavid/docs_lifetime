# Evropa

Evropská část je nejsilnější část kuchařky. Jsou tu české klasiky, italská sýrová jídla, lehké španělské gazpacho, portugalský dezert a řecké hlavní jídlo.

## Země

| Země | Co tu najdete |
| --- | --- |
| Česko | Polévky, sytá hlavní jídla a domácí dezert |
| Itálie | Krémová a sýrová hlavní jídla |
| Španělsko | Studená zeleninová polévka |
| Portugalsko | Sladký dezert |
| Řecko | Jednoduché hlavní jídlo |
